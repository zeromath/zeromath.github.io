[Emacs] Emacs 学习笔记

Posted on 2017-03-15 | Edited on 2019-03-05 | In Emacs

主要列出了一些 Emacs 常用的命令以备查用

缓冲区

作用 命令 函数名
保存当前缓冲区 C-x C-s save-buffer
保存所有缓冲区 C-x s save-some-buffers
列出所有缓冲区 C-x C-b list-buffers
切换缓冲区 C-x b
杀死缓冲区 C-x k kill-buffer

文件

作用 命令 函数名
打开文件 C-x C-f
只读方式打开 C-x C-r
读入另外一个文件代替当前buffer的文件 C-x C-v

文件编辑命令

移动插入点

作用 命令 函数名
缓冲区开始 M-> beginning-of-buffer
缓冲区末尾 M-< end-of-buffer
上一屏 C-v scroll-up
下一屏 M-v scroll-down
段落开头 C-{ beginning-of-paragraph
段落结尾 C-} end-of-paragraph
到指定行 M-g g goto-line
行开头 C-a beginning-of-line
行结尾 C-e end-of-line
句子开头 M-a beginning-of-sentence
句子结尾 M-e end=of-sentence
上一行 C-p previous-line
下一行 C-n next-line
上一个单词 M-b backward-word
下一个单词 M-f forward-word
上一个字符 C-b back-char
下一个字符 C-f forward-char

引用插入、删除、撤销和重复

作用 命令 函数
引用插入 C-q <字符值> quote-insert
删除一个字符 C-d delete-char
删除一个单词 M-d kill-word
撤销 C-_ or C-/ undo
重复 C-u <次数> <命令> universal-argument

标记和复制

作用 命令 函数
删除一行 C-k kill-line
设定标记 C-Space or C-@ set-mark-command
全选 C-x h mark-whole-buffer
选取一页 C-x C-p mark-page
选取一段 M-h mark-paragraph
选取一个单词 M-@ mark-word
选取一页 C-x C-p mark-page
交换标记和光标位置 C-x C-x exchange-point-and-mark
将区域中字母改为大写 C-x C-u upcase-region
删除区块 C-w kill-region
复制区块 M-w kill-ring-save
粘贴 C-y yank

快速复制

作用 命令 函数
将区域中内容加入到一个buffer中 append-to-buffer
将区域中内容加入到一个buffer中，不过加入位置在该buffer的光标前 prepend-to-buffer
将区域中内容加入到一个buffer中，删除该buffer原有内容 copy-to-buffer
在该位置插入指定的buffer中所有内容 insert-buffer
将区域中内容复制到一个文件中 append-to-file

字符修改

作用 命令 函数
光标处单词首字母大写 M-c
光标处单词全部字母大写 M-u
光标处单词全部字母小写 M-l
补全 M-/
交换光标处和之前的字符 C-t
交换光标处和之前的单词 M-t

搜索

作用 命令 函数
从光标处往后搜索 C-s
从光标处往前搜索 C-r
(向下)光标位置的单词作为查找字符串 C-s C-w
(向下)光标位置到行尾作为查找字符串 C-r C-y
非递增查找 C-s RET <查找字符串> RET
不受换行、空格、标点影响 C-s RET C-w
正则式向下查找 C-M-s

用向上查找命令就将上面命令的s替换为r

宏模式

作用 命令 函数
开始一个宏的定义 C-x (
结束一个宏的定义 C-x )
执行宏 C-x e
给最后一个宏命名 name-last-kbd-macro
在当前文件中插入一个已定义并命名过的宏 insert-kbd-macro
