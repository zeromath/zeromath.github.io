发表文章

系统会提示创建了”…/source/_post/你的文章标题.md”。这个MD文件就是我们的文章的主体了，打开这个目录，你会发现里面已经有了”hello-world.md”这个文件。这就是刚刚在浏览器里看到的那个唯一的一篇博文。要删除一篇文章，就直接将对应的MD文件删除即可
