Remarks on Completion of Rings

Posted on 2017-03-09 |

Definition of Completion

Given a ring $R$, not necessarily Noetherian, we could always define its completion $\hat{R}^I$ with respect to an ideal $I\subset R$.

Cauchy Sequences

One way to define this is using Cauchy sequence: A sequence $(r_1,r_2,\cdots,r_n,\cdots)$ is a Cauchy sequence if
$$
\forall n \ \ \exists N \ \text{ such that } \forall i,j\geq N, r_i-r_j\in I^n
$$
while a null sequence $(r_1,r_2,\cdots,r_n,\cdots)$ is a sequence converges to $0$, that is,
$$
\forall n \ \ \exists N \ \text{ such that } \forall i\geq N, r_i\in I^n
$$

The completion $\hat{R}^I$ is defined to be the ring of all Cauchy sequences modulo the ideal generated by null sequences.

Inverse Limit

The completion $\hat{R}^I$ could also be defined as an inverse limit of the system
$$
R/I\leftarrow R/I^2\leftarrow \cdots \leftarrow R/I^n\leftarrow
$$

Completion & Localization

Maximal ideals of $\hat{R}^I$

We have following theorem

Theorem
The kernel of the natural map $\hat{R}^I\rightarrow R/I$ is contained in the Jacobson ideal of $\hat{R}^I$.

Proof
We use the inverse limit construction. So each element $r$ in $\hat{R}^I$ is denoted by a sequence where $(r_1,r_2,…,r_i,…)$ such that $r_i\in R/I^i$ maps to $r_j\in R/I^j$ for any $i>j$.

If $r\in \hat{R}^I$ is in the kernel, then $r={0,r_2,…,r_i,…}$ where each $r_k$ is an element in $I/I^k$. Let $v=(v_1,v_2,…,v_i,…)$ be a sequence such that $v_{i}=1-r_i+r_i^2-\cdots+(-1)^i r_i^i$. Then the image of $v_n$ in $R/I^{n-1}$ is $v_{n-1}$. So $v\in\hat{R}^I$ and we have $(1+r)v=1$. So the kernel is contained in the Jacobson ideal. $\blacksquare$

A direct corollary from this theorem is that we have 1-1 correspondence between maximal ideals of $\hat{R}^I$ and maximal ideals of $R/I$. In particular, if $R/I$ is quasilocal (local but not necessarily Noetherian), then $\hat{R}^I$ is quasilocal.

Completion Commutes with Localization

If $I=Q$ is a maximal ideal of $R$, then $\hat{R}^Q$ is a quasilocal ring. If we localize $R$ at $Q$ and then take its completion, we also get a quasilocal ring $\hat{R_Q}^Q$. In fact, these two rings are isomorphic to each other as we have following lemma

Lemma
If $Q$ is a maximal ideal of $R$, then $R_Q/Q^nR_Q\simeq R/Q^n$

Proof
The only prime ideals in $R/Q^n$ is $Q$ as $Q^n$ is $Q$-primary. So $R/Q^n$ is a quasilocal ring. Therefore the image $R-Q$ is invertible in $R/Q^n$. So we have the desired inverse for the natural map $R/Q^n\rightarrow R_Q/Q^nR_Q$. $\blacksquare$

Vanishing of Sections

Posted on 2017-03-07 |

Original Problem

The problem comes from an exercise (Chapter 1 Ex 3.3(c)) in Hartshorne

Show that if $\varphi(X)$ is dense in $Y$, then the map $\varphi^*_P$ is injective for all $P$.

Here $X$ and $Y$ are assumed to be varieties. But this could be generalized to morphisms of locally ringed spaces.

If $(\varphi,\varphi^\sharp):(X,\mathscr O_X)\rightarrow (Y,\mathscr O_Y)$ is a morphism of locally ringed space, $\varphi(X)$ is dense in $Y$ and $X$ is irreducible, then $\varphi^\sharp_P$ is injective for any $P\in X$

Vanishing of Sections

Before prove this, let’s discuss a little bit about the definition of ‘’section vanishes’’.

Two Definitions

Recall the usual definition of support of a section in a given sheaf $\mathcal F$:

Definition
The support of $s\in\mathcal F(X)$ is $\text{supp}(s)={P\in X|s_P\not=0}$ where $s_P$ is the image of $s$ in the stalk $\mathcal F_P$

The support of a section is always closed because if $P\not\in \text{supp}(s)$, then there is some open subset $U$ containing $P$ such that $s|_U=0$. Therefore $U\cap\text{supp}(s)=\emptyset$, which shows that $\text{supp}(s)$ is closed.

However, recall the definition of the support of a function, it’s the preimage of $k^*$, which, in particular, is open. If we are considering the sheaf of functions here, then what’s the ‘’correct’’ definition of support of a section (which is a function) corresponding to the definition of support of a function?

If you think about this, the answer seems to be immediate. A function $f$ vanishes at a point $P$ when its function value is zero at that point. Think about the definition of the stalk at that point, the image of $f$ has to lie in the maximal ideal of the local ring $\mathcal F_P$. This enlightens us to give following defintion

Definition
The support of $s\in\mathcal F(X)$ as a function is $\text{fupp}(s)={P\in X|s_P\not\in m_{X,P}}$ where $m_{X,P}$ is the maximal ideal in the stalk $\mathcal F_P$

Topological Properties

We already saw that $\text{supp}(s)$ is closed. It turns out that $\text{fupp}(s)$ is open. Because if $P\in\text{fupp}(s)$, then $s_P$ is not in the maximal ideal $m_{X,P}$, so it has an inverse $(t,U)$. That is, $st|_U=1$. So they are inverse to each other in any stalk at the points in $U$. Therefore $p\in U\in \text{fupp}(c)$.

It’s also clear from the definition that $\text{fupp}(s)\subset\text{supp}(s)$. For notational reason, let’s denote the complement of $\text{supp}(s)$ (resp. $\text{fupp}(s)$) by $\text{szero}(s)$ (resp. $\text{fzero}(s)$).

Then we already established

  • $\text{fupp}(s)$ is open and contained in the closed subset $\text{supp}(s)$
  • $\text{szero}(s)$ is open and contained in the closed subset $\text{fzero}(s)$

The behaviour of the functional support and zero locus is very similar to the behaviour of functions we know. We have following theorems

Theorem
If $(\varphi,\varphi^\sharp):(X,\mathscr O_X)\rightarrow (Y,\mathscr O_Y)$ is a morphism of locally ringed spaces, then
$$
\varphi^{-1}(\text{fupp}(s))=\text{fupp}(\varphi^\sharp(s))
$$

It’s immediate from the theorem that $$\varphi^{-1}(\text{fzero}(s))=\text{fzero}(\varphi^\sharp(s))$$ Meanwhile, the behaviour of section is not that good

Theorem
If $(\varphi,\varphi^\sharp):(X,\mathscr O_X)\rightarrow (Y,\mathscr O_Y)$ is a morphism of locally ringed spaces, then
$$
\varphi^{-1}(\text{szero}(s))\subset\text{szero}(\varphi^\sharp(s))\subset \varphi^{-1}(\text{fupp}(s))=\text{fupp}(\varphi^\sharp(s))
$$

The proof of these two theorems comes from the commutative diagram
$\require{AMScd}$
\begin{CD}
\mathscr O_Y(U) @>>> \mathscr O_X(\varphi^{-1}(U)) @. s @>>> \varphi^\sharp (s) \\
@VVV @VVV @VVV @VVV \\
\mathscr O_{Y,\varphi(P)} @>>> \mathscr O_{X,P} @. s_{\varphi(P)} @>>> \varphi^\sharp(s)_P
\end{CD}

If $s_{\varphi(P)}$ is zero (resp. contained in the maximal ideal), then $\varphi^\sharp(s)$ maps to zero (resp. contained in the maximal ideal). So $P\in\text{szero}(\varphi^\sharp(s))$ (resp. $P\in\text{fzero}(\varphi^\sharp(s))$). For th converse direction, if $s_{\varphi(P)}$ is not contained in the maximal ideal), then $\varphi^\sharp(s)$ maps to a unit. So $P\in\text{fupp}(\varphi^\sharp(s))$

Proof of Original Problem

Now we have enough tools to prove our original problem

Proof
For any point $P\in X$, let $(f,U)$ be a nonzero element in the stalk $\mathscr O_{Y,\varphi(P)}$. Replace $Y$ with $U$, then $f\neq 0$ means that $\varphi(P)\in\text{supp}(f)$. We want to prove that $(\varphi^\sharp(f),\varphi^{-1}(U))$ is not zero in the stalk $\mathscr O_{X,P}$. That is equivalent to $P\in\text{supp}(\varphi^\sharp(f))$

Denote $\varphi^\sharp(f)$ by $g$. Assume that $g\not\in\text{supp}(g)$, then since $\text{supp}(g)$ is closed, there is some open subset $V$ such that $p\in V\subset \text{szero}(g)$.

On the other hand, $\varphi(X)$ is dense in $Y$, so it’s dense in $U$. Therefore $\varphi(X)\cap \text{fupp}(f)\neq \emptyset$. So there is some point $Q$ such that $\varphi(Q)\in\text{fupp}(g)$. But $\text{fupp}(g)$ is open, so there is some open subset $V’$ such that $Q\in V’\subset \text{fupp}(g)$.

Note that $X$ is irreducible, $V\cap V’\neq\emptyset$, but this is a contradiction as $\text{fupp}(g)\cap\text{fzero}(g)=\emptyset$ and $\text{szero}(g)\subset \text{fzero}(g)$. $\blacksquare$
